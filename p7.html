함수

  • C언어에서 함수의 의미와 역할을 설명할 수 있다.
  • 사용자 정의함수를 정의하고 사용할 수 있다.

함수 정의

함수 : 특정한 작업을 수행하는 문장을 모아둔 단위 (라이브러리 함수, 사용자 정의 함수)

함수 장점

  1. 코드가 중복되지 않고 간결함
  2. 코드의 재사용성이 증가함
  3. 프로그램의 모듈화가 증대됨
  4. 유지보수가 용이함

함수 정의 형식

        리턴형 함수명 (데이터명 매개변수명 [, 데이터형 매개변수명, ..])
        {
            문장;
        }

리턴형

  1. 함수는 처리 후 결과를 함수를 호출한 곳으로 값을 보내줄 수 있음
  2. 리턴되는 값의 데이터 타입이 함수의 리턴형
  3. void, char, int, float...

매개변수(Argument)

  1. 함수 호출 시 입력값을 전달하는 역할
  2. ","로 구분하여 여러 개 전달 가능
  3. 매개변수가 없는 경우는 void

함수 선언 및 호출

선언

  1. 함수는 사용 전 반드시 선언해야 함
  2. 선언은 함수의 원형(Prototype)을 기술함
  3. 선언 시 매개변수 이름은 생략 가능함
  4. 함수 호출 전에 정의문이 먼저 기술된 경우는 생략 가능함

호출

  1. 함수를 실행함
  2. 함수 이름과 매개변수를 기술함
  3. 호출 시 해당 함수에 정의된 문장이 수행됨
