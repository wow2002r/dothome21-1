변수와 상수

1. 변수

1) 변수란?

변수 : 프로그램 수행 시 데이터를 보관할 수 있는 공간
(1) 메모리에 저장공간 확보
(2) 방법에 따라 크기와 용도가 다르게 생성
(3) 유일한 이름(Identifier) 부여 (4) 메모리에 대한 심볼릭 표현

2) 변수선언

변수는 사용 전 반드시 먼저 선언해야 함
필요 시 선언과 동시에 초기화도 가능함
변수선언문 기본 형식 : 데이터타입 변수명; / 데이터타입 변수명1, 변수명2, ......;
변수명은 식별자 생성 규칙을 적용함

3) 식별자

식별자(Identifier) : 변수를 구분하기 위해서 사용되는 이름
식별자 규칙
ㄱ. 식별자는 반드시 영문자, 숫자, 밑줄 기호(_)만을 사용해야 함
ㄴ. 식별자의 첫 글자는 반드시 영문자 또는 밑줄 기호(_)로 시작해야 함(식별자는 숫자로 시작해서는 안됨)
ㄷ. 식별자에는 밑줄 기호(_)를 제외한 다른 기호를 사용할 수 없음
ㄹ. 식별자는 대소문자를 구분해서 만들어야 함
ㅁ. C언어의 예약어는 식별자로 사용할 수 없음( if and printf switch case break continue int char float while for double return ...)

변수 활용 : 초기화는 =(대입연산자)를 이용함
변수명 = 값 ;

2. 상수(Consonant)

1) 상수란?

상수 : 변하지 않는 자료값. 보관되는 값에 따라 의미가 변하는 것이 상수
리터럴 상수(Literal Consonant) : 정수형, 실수형, 문자형
심볼릭 상수(Sumbolic Consonant) : const 상수, 매크로 상수, 열거형 상수

매크로 상수 : #define 문으로 정의 되는 상수

형식 : # define 매크로이름 상수

데이터타입

1. 개요

C 프로그램에서 사용되는 모든 변수나 상수 값은 정해진 데이터형을 가짐
데이터형에 의해서 주어진 값의 이진 표현이 결정됨

기본 데이터형(Primitive Data Ytpe) : 문자형(char), 정수형(short, int, long, long long), 실수형(float, double, long double)
파생 데이터형(Derived Data Type) : 배열, 포인터 사용자 정의형(User-defined Data Type) : 구조체, 공용체, 열거체

2. 문자형

char : signed char 을 의미. 1byte.
unsigned char : 부호 없는 문자형을 의미. 1byle.(br)

문자형 변수 선언
char 변수명;
char 변수명, 변수명, ... 변수명;
char 변수명 = '초기값';

3. 정수형

1) 정수형의 개념 : C언어는 정수형으로 short, int, long을 제공함

크기 : short <= int <= long
정수형 : signed형(생략 가능), unsigned형

short(signed short int), unsigned short(unsigned short int) : 2byte
int(signed int), unsigned int(unsigned int) : 4byte
long(signed long int), unsigned long(unsigned long int) : 4byte
long long(signed long long int), unsigned long long(unsinged long long int) : 8byte

2) 정수의 표현

(1) 부호가 있는 정수형은 최상의 비트(msb)를 부호 비트로 사용함
(2) 부호가 있는 정수형은 음수를 펴현하는데 2의 보수를 사용함
(3) 부호가 없는 정수형은 최상위 비트를 값을 저장하는 용도로 사용함

4. 실수형

1) 실수의 표현 방식 : 고정소수점(Fixed Point)방식, 부동소수점(Floating Point)방식
2) 실수의 저밀도 : 단정도(Float)형, 배정도(Double)형

5. 형식지정자

1) 형식지정자를 이용한 printf() 출력 : %c(문자), %d(정수값), %o(8진수) 등
